1. Core Syntax

Marker Keyword Description
📦 package Namespace: Declares the operational scope of the file.
🔌 use Import: Loads engines or providers like aina, edr, aws, and threat.
⚙️ keep Constant: Defines thresholds, modes, and fixed configuration values.
⭕️ define Open Circle Block: Starts a reusable helper function block.
🔥 check Entry Point: Starts an active monitoring or execution routine.
⚠️ when Conditional Gate: Opens a condition block.
🔹 set Assignment: Stores runtime values such as hostnames, users, processes, or scores.
🌀 change Mutation: Updates an existing value.
⚡ do Action Block: Opens the live execution zone.
⭕️ each Open Circle Block: Starts an iterator block.
↩️ return Output: Returns helper data or results.
☁️ provider action Execution Call: Invokes real cloud, EDR, Linux, IAM, or API actions.
🚨 alert Signal: Raises an alert for analysts or dashboards.
📣 notify Communication: Sends messages to Teams or other channels.
🧾 evidence Forensic Proof: Records artifacts, timestamps, and outcomes.
⛓️ rosecoin Immutable Provenance: Anchors evidence to Rosecoin.
✨ gui Visual Update: Pushes state into the visual interface layer.
🔴 close block Close Circle Block: Closes define, each, when, do, and check blocks.
